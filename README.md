# DentVision SaaS - Стоматологическая CRM система

Полнофункциональная CRM для стоматологических клиник с 3D одонтограммой, фотопротоколом и автоматизацией.

## 🚀 Функции

### Для клиники:
- **3D Odontogram** — поверхности зубов (M, O, D, B, L), цветовая маркировка, авто-план лечения
- **Фотопротокол** — категории (улыбка, лицо, интраоральные), сравнение До/После, рисование
- **Расписание** — Drag & Drop, цветные категории пациентов, WhatsApp подтверждение, напоминания, лист ожидания
- **Финансы** — долги, предоплата, рассрочка, Kaspi, прибыль врача/клиники, расходы, зарплата, склад материалов
- **Лаборатория** — контроль сроков, цифровой Wax-Up, Smile Design

### Для руководителя:
- **Dashboard** — прибыль за день/месяц, загрузка врачей, средний чек, конверсия, отмены, неявки, топ услуг, эффективность рекламы

### Системные функции:
- **Регистрация клиник** — автоматическая с выбором тарифа
- **Подписки** — Basic (бесплатно), Pro (9900₸), Enterprise (24900₸)
- **Выход из аккаунта** — кнопка в sidebar
- **GitHub интеграция** — ссылка на репозиторий

## 📁 Структура проекта

```
src/
├── App.jsx                    # Главный компонент с навигацией
├── index.jsx                  # Точка входа
├── utils/
│   ├── constants.js           # Константы, дизайн-токены, статусы
│   └── supabase.js            # CRUD операции, WhatsApp, подписки
├── hooks/
│   └── useData.js             # 8 кастомных хуков для данных
├── components/
│   ├── ui/
│   │   └── BaseComponents.jsx # 12 базовых UI компонентов
│   └── Odontogram3D.jsx       # 3D зубная формула с поверхностями
└── pages/
    ├── Login.jsx              # Вход/регистрация, подписки, GitHub
    ├── Dashboard.jsx          # Метрики, загрузка врачей, топ услуг
    ├── Schedule.jsx           # Drag&Drop расписание, лист ожидания
    ├── Patients.jsx           # 3D odontogram, фотопротокол, сравнение
    ├── Cashier.jsx            # Финансы, долги, Kaspi, склад, зарплата
    ├── Lab.jsx                # Лаборатория, Wax-Up, Smile Design
    ├── AITeam.jsx             # AI ассистенты
    └── SuperAdmin.jsx         # Управление клиниками, подписками
```

## 🛠 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build
```
